# Krz-Tech Minecraft Server Project
## Coderクローン用リポジトリ
このリポジトリは、Krz-Tech Minecraftサーバー開発プロジェクトの開発環境として使用されるCoderワークスペースにクローンされる初期リポジトリです。

## 関連リポジトリ
- [Krz-Tech/minecraft-project](https://github.com/Krz-Tech/minecraft-project): メインのMinecraftサーバープロジェクトリポジトリ
- [Krz-Tech/minecraft-project-skript](https://github.com/Krz-Tech/minecraft-project-skript): Minecraftサーバー用のSkriptコードを管理するリポジトリ
- [Krz-Tech/coder](https://github.com/Krz-Tech/coder): Krz-TechのCoderワークスペースのテンプレートを管理するリポジトリ
